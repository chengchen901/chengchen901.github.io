常量、变量、数据类型与占位符

常量

单个常量的定义方式：const 常量名=值

多个常量的定义方式：const (常量名=值
常量名=值
常量名=值
……
)

注意：常量一旦确认，无法修改

一般来说常量都是大写，常量定义在 main 函数内是局部常量，放在 main 函数外是全局常量

变量

传统定义方式：var 变量名 类型=值

自动推导类型定义方式：变量名:=值

注意：自动推导类型只适用于局部变量，无法适用于全局变量

常用的基本数据类型

布尔 bool

整形 int

浮点 float32、float64（变量名:=浮点数 默认类型是 float64）

字符 byte

字符串 string

输出与占位符

fmt.Printf(“占位符”, 变量)

%T：输出变量类型

%x：以16进制打印

%d：以整形的方式输出结果

%a：打印变量内存地址，例如fmt.Printf(“%a”, &变量)

%s：以字符串打印结果
