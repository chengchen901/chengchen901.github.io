Golang

开发环境介绍

如何安装 Golang

https://studygolang.com/dl 在该网址中选择合适的版本进行下载。

完成后在 cmd 中输入命令：go env

出现如下提示则说明安装完成

IDE（Integrated Development Environment） 的选择

能够进行 Golang 开发的 IDE 有那些？

文本编辑器、sublime、vim、notepad++、eclipse、liteIDE、vscode、Goland。

初识 Golang 代码

GOPATH 为开发基础目，如果需要变更，修改系统环境变量即可，在 cmd 输入 go env 查看 GOPATH 是否变更

我自己的 GOPATH 为：D:\workspace\GoWork，GoWork 是我自己创建的目录，完整的 GoWork 目录下面有 bin、pkg、src 目录

src：主要是存放我们的源代码

bin：代码编译后的二进制文件

pkg：存放网上下载的第三方包

在 src 目录下面新建 helloworld 文件夹，使用 IDE 打开 helloworld 文件夹，新建 helloworld 文件。

Golang 是一门静态、编译型语言，所以它跟 Java 一样需要一个 main 函数入口，main 函数所在的 package 必须为 main，所以新建 helloworld 文件之后先把 package helloworld 修改为 package main。再编写 main 函数打印 hello world。代码如下：
